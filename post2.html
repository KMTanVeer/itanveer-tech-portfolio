ESP32 WiFi Setup Quick Guide

August 8, 2025 7 min read ESP32, IoT

The ESP32 is a powerful microcontroller with built-in WiFi and Bluetooth capabilities, making it perfect for IoT projects. In this guide, I'll show you how to set up WiFi connectivity on your ESP32 and explore various configuration options.

Prerequisites

  • ESP32 development board
  • Arduino IDE with ESP32 board package
  • USB cable for programming
  • WiFi network credentials

Basic WiFi Connection

Let's start with a simple WiFi connection example:

#include <WiFi.h>

const char* ssid = "YOUR_WIFI_NAME";
const char* password = "YOUR_WIFI_PASSWORD";

void setup() {
  Serial.begin(115200);
  delay(1000);
  
  Serial.println("Connecting to WiFi...");
  WiFi.begin(ssid, password);
  
  while (WiFi.status() != WL_CONNECTED) {
    delay(1000);
    Serial.print(".");
  }
  
  Serial.println();
  Serial.println("WiFi connected!");
  Serial.print("IP address: ");
  Serial.println(WiFi.localIP());
}

void loop() {
  // Your main code here
}

Advanced WiFi Configuration

For more robust applications, consider these advanced features:

1. Connection Status Monitoring

void checkWiFiConnection() {
  if (WiFi.status() != WL_CONNECTED) {
    Serial.println("WiFi connection lost. Reconnecting...");
    WiFi.begin(ssid, password);
    
    int attempts = 0;
    while (WiFi.status() != WL_CONNECTED && attempts < 20) {
      delay(500);
      Serial.print(".");
      attempts++;
    }
    
    if (WiFi.status() == WL_CONNECTED) {
      Serial.println("\nReconnected to WiFi");
    } else {
      Serial.println("\nFailed to reconnect");
    }
  }
}

2. Multiple Network Support

struct WiFiNetwork {
  const char* ssid;
  const char* password;
};

WiFiNetwork networks[] = {
  {"Home_WiFi", "home_password"},
  {"Office_WiFi", "office_password"},
  {"Mobile_Hotspot", "mobile_password"}
};

void connectToAvailableNetwork() {
  int numNetworks = sizeof(networks) / sizeof(networks[0]);
  
  for (int i = 0; i < numNetworks; i++) {
    Serial.print("Trying network: ");
    Serial.println(networks[i].ssid);
    
    WiFi.begin(networks[i].ssid, networks[i].password);
    
    int attempts = 0;
    while (WiFi.status() != WL_CONNECTED && attempts < 10) {
      delay(1000);
      attempts++;
    }
    
    if (WiFi.status() == WL_CONNECTED) {
      Serial.println("Connected successfully!");
      return;
    }
  }
  
  Serial.println("Failed to connect to any network");
}

WiFi Modes

The ESP32 supports different WiFi modes:

Station Mode (STA)

Connects to an existing WiFi network:

WiFi.mode(WIFI_STA);
WiFi.begin(ssid, password);

Access Point Mode (AP)

Creates its own WiFi network:

WiFi.mode(WIFI_AP);
WiFi.softAP("ESP32_Hotspot", "password123");

Dual Mode (AP + STA)

Acts as both client and access point:

WiFi.mode(WIFI_AP_STA);
WiFi.begin(ssid, password);
WiFi.softAP("ESP32_Hotspot", "password123");

Useful WiFi Functions

  • WiFi.localIP() - Get assigned IP address
  • WiFi.RSSI() - Get signal strength
  • WiFi.macAddress() - Get MAC address
  • WiFi.disconnect() - Disconnect from network
  • WiFi.scanNetworks() - Scan for available networks

Troubleshooting

Common issues and solutions:

  • Connection fails: Check SSID and password spelling
  • Weak signal: Move closer to router or use external antenna
  • Frequent disconnections: Implement auto-reconnect logic
  • Slow connection: Check for interference on 2.4GHz band

Best Practices

  • Always check connection status before transmitting data
  • Use WiFiManager library for dynamic configuration
  • Implement connection timeout and retry logic
  • Store credentials securely (consider EEPROM or SPIFFS)
  • Monitor power consumption in battery-powered projects
