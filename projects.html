Arduino Digital Multimeter

A precision digital multimeter built using Arduino Uno, capable of measuring DC/AC voltage, current, and resistance with high accuracy. Features include LCD display, auto-ranging, and data logging capabilities.

Goal: Create a cost-effective, accurate digital measuring instrument
Tools Used: Arduino IDE, Proteus simulation, KiCad for PCB design
Challenges: Achieving high precision, calibration, noise reduction
Results: ±1% accuracy for voltage, ±2% for current measurements

Distance Protection with MATLAB Simulink

Advanced power system protection scheme implemented in MATLAB Simulink for transmission line fault detection and isolation. Includes impedance calculation, fault classification, and trip signal generation.

Goal: Develop reliable transmission line protection system
Tools Used: MATLAB Simulink, Power Systems Toolbox, Scope analysis
Challenges: Fault classification, setting coordination, CT/PT modeling
Results: 98% fault detection accuracy, <100ms trip time
